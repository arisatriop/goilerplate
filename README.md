# Goilerplate

Goilerplate is a starter template for Go projects, designed to help you quickly set up a new Go application with best practices and a clean project structure.

## Tech Stack

- Golang : https://github.com/golang/go
- Postgres : https://github.com/jackc/pgx/v5
- MySQL : https://github.com/mysql/mysql-server
- Redis : https://github.com/redis/go-redis/v9

## Framework & Library

- GoFiber (HTTP Framework) : https://github.com/gofiber/fiber
- GORM (ORM) : https://github.com/go-gorm/gorm
- Go Playground Validator (Validation) : https://github.com/go-playground/validator
- Logrus (Logger) : https://github.com/sirupsen/logrus
- Viper (Configuration) : https://github.com/spf13/viper

## Configuration

All configuration is in `config.yaml` file.
